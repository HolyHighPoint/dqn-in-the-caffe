# QLearning

Human-Level Control Through Deep Reinforcement Learning

## Compiling

    ALE_HOME=... CAFFE_HOME=.. ./configure

For example:

    ALE_HOME=~/Workspace/ale_0_5 CAFFE_HOME=~/Workspace/caffe ./configure
    cd build
    make

## Usage

    ./qcli learn
    ./qcli evaluate
